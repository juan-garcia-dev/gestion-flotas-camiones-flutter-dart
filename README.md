# Gestión de Flota de Camiones

Proyecto desarrollado en Flutter.

## Tecnologías utilizadas:
* **Persistencia:** SQLite para el almacenamiento local.
* **Gestión de Estado:** Provider para la lógica de negocio.
* **UI:** Widgets nativos de Flutter y FutureBuilders para carga asíncrona.

## Funcionalidades:
* Alta y baja de camiones.
* Filtrado dinámico por asociaciones.
